От object к iframe — другие технологии встраивания

Пример

Теги

  • iframe - предназначен для встраивания других веб-страниц

Атрибуты iframe

  • src - путь до видео
  • title - название встраиваемого плеера
  • frameborder - если установлено значение 1, это указывает браузеру нарисовать границу между этим фреймом и другими фреймами, что является поведением по умолчанию. 0 удаляет границу
  • allow - разрешения на действия
  • allowfullscreen - если установлено, iframe может быть помещён в полноэкранный режим с использованием полноэкранного API
  • width - ширина видео
  • height - высота видео
