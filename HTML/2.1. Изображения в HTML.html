Пример

Пояснения

Теги

  • img - вставка изображения в документ
  • figcaption - название изображения
  • figure - блок для изображения

Атрибуты img

  • src - путь до изображения. Может быть в виде ссылки или пути до файла
  • alt - текстовая замена изображения в случаях, когда картинка не появилась по какой-то причине
  • title - текстовое описание изображения при наведении курсора на него
  • width - ширина изображения
  • height - высота изображения
