Теги

  • caption - название таблицы
  • thead - обертка для заголовка таблицы
  • tbody - обертка для тела таблицы
  • tfoot - обертка для окончания таблицы

Атрибуты

  • th.scope - определение ячейки, как заголовка (col, row, colgroup и rowgroup)
  • th.id - идентификатор ячейки
  • td.headers - каждый headers атрибут должен содержать список всех id , разделённый пробелами, ко всем th элементам которые действуют как заголовок для этой ячейки.
