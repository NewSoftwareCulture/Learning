Начало работы с HTML

Тег «p» используется для выделения параграфов

Тег «i» добавляет курсив

Тег «u» добавляет подчеркивание

Тег «b» добавляет жирный

Тег «em» добавляет курсив

Тег span предназначен для определения строчных элементов документа

Тег «strong» выделяет текст жирным

Тег «img» с атрибутом src используется для добавления изображений на страницу

Тег «a» обрачивает элемент в ссылку. Имеет атрибуты: href (ссылка), title (текст-подсказка), target (контекст просмотра)

Тег «input» представляет из себя форму ввода. Также имеет булевый атрибут "disabled"
