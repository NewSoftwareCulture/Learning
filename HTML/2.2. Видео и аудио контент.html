Видео и аудио контент

Теги

  • video - вставка видео в документ
  • track - вставка субтитров к видео
  • audio - вставка аудио в документ
  • source - вставка ресурса в документ

Атрибуты video

  • src - путь до видео
  • controls - включение нативных контроллеров воспроизведения видео
  • autoplay - позволяет сразу начать воспроизведение видео, пока остальная часть страницы загружается
  • loop - позволяет воспроизводить видео снова, когда он заканчивается
  • muted - заставляет проигрыватель воспроизводить звук, отключённый по умолчанию
  • poster - принимает в качестве значения URL-адрес изображения, который будет отображаться до воспроизведения видео (заставка)
  • preload - используется в элементе для буферизации больших файлов
  • width - ширина видео
  • height - высота видео

Атрибуты track

  • src - путь до файла с субтитрами
  • kind - указывает, являются ли реплики субтитрами, титрами или описаниями
  • srclang - установка языка субтитров

Атрибуты audio

  • src - путь до аудио
  • controls - включение нативных контроллеров воспроизведения видео и аудио
  • autoplay - позволяет сразу начать воспроизведение аудио, пока остальная часть страницы загружается
  • loop - позволяет воспроизводить аудио снова, когда он заканчивается
  • muted - заставляет проигрыватель воспроизводить аудио, отключённый по умолчанию
  • preload - используется в элементе для буферизации больших файлов
